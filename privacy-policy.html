Privacy Policy

1. Introduction

Mark Buddy is a Chrome extension designed to help you organize and manage your bookmarks. This privacy policy explains how we handle your data and what information we access.

2. Data Collection and Storage

Important: Mark Buddy does not collect, store, or process any data on external servers. All data is stored locally in your browser using Chrome's built-in storage and sync capabilities.

We access the following data, all of which is stored locally in your browser:

  • Your existing bookmarks (stored in Chrome's native bookmark system)
  • Custom categories and organization preferences (stored in Chrome's sync storage)
  • UI preferences and settings (stored in Chrome's local storage)

3. How We Use Your Data

Your data is used exclusively for:

  • Displaying and organizing your bookmarks
  • Managing custom categories
  • Enabling bookmark export and import
  • Storing your preferences

We do not:

  • Collect personal information
  • Track browsing behavior
  • Share data with third parties
  • Use data for advertising
  • Store data on external servers

4. Data Storage and Sync

All data is stored in your browser using Chrome's native storage systems:

  • Bookmarks are stored in Chrome's bookmark manager
  • Settings and preferences are stored in Chrome's sync storage
  • If you enable Chrome sync, your data will be synced across your devices using Chrome's built-in sync feature

5. Permissions We Use

Mark Buddy requires the following permissions, all of which are used locally:

  • bookmarks: To access and manage your bookmarks
  • storage: To save your preferences and settings
  • tabs: To access current tab information when adding bookmarks
  • activeTab: To get current page information when you add a bookmark
  • notifications: To show local notifications for actions
  • contextMenus: To add right-click menu options for quick bookmarking

6. Your Control Over Data

You have complete control over your data:

  • Export your bookmarks at any time
  • Delete the extension to remove all local data
  • Control sync settings through Chrome
  • Modify or delete bookmarks at any time

7. No Remote Code

Mark Buddy does not use any remote code. All functionality is contained within the extension package, and we do not:

  • Load external scripts
  • Use remote modules
  • Execute code from external sources
  • Use eval() or similar functions

8. Changes to This Policy

We may update this privacy policy from time to time. Any changes will be reflected in the "Last updated" date at the top of this page.
